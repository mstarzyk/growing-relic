# What is it
Experiments with Rails and nginx.

# What is required
- Docker
- GNU Make

# How to run
1. Build docker image:
```
make build-docker
```
2. Run docker
```
make run
```

# Does the project name mean anything?
No. It comes from:

```
xkcd-pass -n 2
```

See [xkcd-pass](https://github.com/redacted/XKCD-password-generator).
